# chatbox

## Lancer le programme

1. Cloner ce repo
2. ouvrir un terminal et se rendre à l'emplacement du repo et executer
   `python3 server.py`
3. Puis :
   `python3 client.py`

à noter que l'ordre de ces 2 commande est a respecté.
